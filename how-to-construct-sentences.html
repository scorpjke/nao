Nao textbook: How to construct sentences

The sentence consists of top-words. Mostly they are marked by markers that come right before to inform about the function of the word in the sentence.

Nao sentence can have the following top-words:
1. Top-verb (= predicate = main verb). Marked with ho in case of non-standard word order.
2. Top-nouns:
a) Top-subject. Must come before the top-verb unless marked with gi.
b) Top-object. Must come after the top-verb unless marked with gu.
c) Nouns that follow de or [po+verb/preposition].
3. Adjective (= top-noun modifier). Marked with ce if not originally an adjective. Must follow the top-noun.
4. Top-adverb (= top-verb modifier). Always marked with po.
5. Clause (= subsentence). Optionally marked by pe or words ca, kei/keu, sei/seu, hoi/hou.
Any top-word (including subject, object, and even the main verb) can be omitted if the speaker wishes to do so, as long as the grammatical structure of the sentence remains clear.

Example 1:
— miau gin lias —
Cat eats fish.
At first we find the top verb. In this case it is gin = eat/drink, and the tense is not specified. We don’t see any more markers in the sentence so the subject is on the left: miau = cat, and the object is on the right: lias = fish. The resulting meaning is “Cat eats fish”.

Example 2:
— megau pojoi kigo pua kan latuadue pun ceniegau —
We happily went to see big annual competition.
1. Find the top verb. It's kigo pua kan [PAST go for see/look] = went to see. As you can see, it consists of a few “words”, but in Nao it is all grammatically one top verb.
2. Find the subject. It should be somewhere on the left of the main verb. At first we see pojoi [ADV happy] = happily, which is marked with po - the top-adverb marker. Let’s look more to the left and we see megau not marked with anything so it must be the subject.
3. We see an unmarked noun coming after the main verb so it’s an object: latuadue.
4. pun=big is an adjective by default, stand-alone word, so it is a top-adjective describing the top-noun right before it: latuadue.
5. niegau = every year, is a noun, but then it is marked by ce so it becomes a top-adjective describing the top-noun on the left.

Other rules

1. Top-words must be divided by a space. A marker can be thought of as a part of the top-word and thus does not have to be separated from it by a space.
2. Words [din=and, dun=or] can only connect top-words or sentences. If a word that follows din/dun is not marked, it is the same type of top-word as the one right before din/dun. To say "and" inside a compound, use ko.
