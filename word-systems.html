Word systems

  1. Expressing numbers
    1. Numbers are expressed by reading out its digits from left to right.
    2. The number is placed before the noun to describe the number of objects.
    3. To express decimal fractions, use the word dot: tuidotjafos = 3.14
    4. To express big and little fractional numbers, one can use scientific notation: jadotfospet puset (= 1.45pu7) means 1.45*107 or 14,500,000
    5. Negative numbers are expressed using has: hasdua = -2 (negative two)
  2. Colors can be expressed describing the hue, luminosity and saturation (see the color wheel):
    1. The full color spectrum is defined on the scale [0;6): ha0=red, ha1=yellow, ha2=green, ha3=cyan, ha4=indigo blue, ha5=magenta.
    2. Luminosity is expressed with lai/lau=light/dark plus an optional number from 0 to 9. If the luminosity is not mentioned at all, it is in range of lau5 and lai5. Otherwise, if the value is not mentioned, it is in range of [5,10)
    3. Saturation is expressed with bul/bil=strong/weak plus an optional number from 0 to 9. If the saturation is not mentioned at all, it is in range of bil5 and bul5. Otherwise, if the value is not mentioned, it is in range of [5,10).
    4. Then we can add extra digits (0-9) to make the parameter more specific: ha00=pure red, ha05=orange, ha07=gold, etc. Same with luminosity and saturation.
  3. Using numbers with words for relatives:
    1. pau2 = paupau, pau3 = paupaupau, etc. Same for pai
    2. kio2 = paikiopau, kio3 = pai2kiopau2, etc.
    3. kio without any number only refers to kio1. Same for pau.
    4. The word tia is used to refer to the family of one's parent's spouse: mupautia = step-father, mipauduatia = step-grandmother (step-parent's mother), etc.
  4. buten=arm is divided into parts from 0 to 5 (buten0...buten5, from the shoulder to the hand, respectively). Same for bugo=leg.
  5. Words for dishes are made by listing the ingridients (from most to least important). Way of cooking can be mentioned at the end. For example, mespottomfes implies that meat is the biggest/most important ingridient and tomatoes are the smallest/least important, and all of that has been fried together. Use ko to show that things have been cooked separately. For example, potkoliasfes implies that potatoes and fish have been cooked separately: fish has been fried while it is not mentioned how exactly the potatoes have been cooked.
  6. Use tam=atom and numbers to make the words for chemical elements (by referring to the number of protons). For example, tamsetnef = tam79 = gold
