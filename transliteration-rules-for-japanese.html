Transliteration rules for Japanese

Here are the non-obvious links between the Japanese syllables and Nao transliteration:

Japanese 　Nao　 Example
し、しゃ、しゅ、しょ ci, ca, cu, co 四国 → しこく → 'cikoku
ち、ちゃ、ちゅ、ちょ tci, tca, tcu, tco 千葉 → ちば → 'tciba
じ、じゃ、じゅ、じょ ji, ja, ju, jo 富士 → ふじ → 'fuji
ざ、ず、ぞ、ぜ za, zu, zo, ze 銀座 → ぎんざ → 'ginza
ふ、ふぁ、ふぇ、ふぉ、ふぃ fu, fa, fe, fo, fi ファッション → 'faccon
ら行 r スバル → 'subaru
つ tsu コタツ → 'kotatsu
わ ua 川崎 → かわさき → 'kauasaki
や/ゃ、よ/ょ、ゆ/ゅ、いぇ ia, io, iu, ie ヤマハ → 'iamaha
ou or oo as a part of the same kanji/root o 北海道 → ほっかいどう → 'hokkaido
大阪 → おおさか → 'osaka
uu as a part of the same kanji/root u 九州 → きゅうしゅう → 'kiucu
ん n 群馬 → ぐんま → 'gunma
ー (長音記号) (ignored) セーラー服 → 'serafuku
っ repeat the [first] consonant 札幌 → さっぽろ → 'sapporo
抹茶 → まっちゃ → 'mattca
