Nao textbook: How to construct words

The basis of the language are the so-called elementary words.
The structure of each standard elementary word: one consonant + vowel(s) + (optionally) consonant(s).
For example, ku, sim, faia are elementary words but bombufui is a compound word consisting of bom + bu + fui.

Nouns and verbs

Two main groups of elementary words are nouns and verbs, and here are the rules of how to put them together. Any elementary noun or verb can be modified (=described / made more specific) by another noun or verb: just put the describing word after the word being described. Note that the left word must be elementary.
Compound Primary bond
[noun1] [noun2] [noun1] of [noun2]
[noun] [verb] [noun] who does [verb]
[verb] [noun] [verb], the object being [noun]
[verb1] [verb2] [verb1], the object being [verb2]
Secondary bond
[word1] described by [word2]
If the left word is a noun, the resulting compound is a noun.
If the left word is a verb, the resulting compound is a verb.
Use the secondary meaning if and only if the primary meaning doesn't make much sense.

Let’s consider each of the four cases in detail:

Case 1: [noun1] [noun2]

a) [noun1] of [noun2], for example:
cu = top
likbac = face
culikbac = forehead (“top“ of “face“)
b) [noun1] described by [noun2], for example:
dia = day
biu = city/town/village
diabiu = city day (“day“ described by “city“)

Case 2: [noun] [verb]

a) [noun] who does [verb], for example:
da = person
banbedue = win
dabanbedue = winner (“person“ who “wins“)
b) [noun1] described by [verb2], for example:
fue = cloth
neleu = make dry
fueneleu = wiping cloth (“cloth” described by “make dry”)

Case 3: [verb] [noun]

a) [verb], the object being [noun], for example:
fi = like
miau = cats
fimiau = like cats (“like“, the object being “cats“)
b) [verb] described by [noun], for example:
go = go
bu = bodypart
gobu =
go using only one's own body force (“go“, described by “bodypart“)

Case 4: [verb1] [verb2]

a) [verb1], the object being [verb2], for example:
moi = want
sa = know
moisa = want to know (“want“, the object being “know“)
b) [verb1] described by [verb2], for example:
fo = believe/be convinced
sim = be similar/same
fosim = agree (“believe/be convinced“ described by “be similar/same“)

Prefixes

Prefixes are a relatively small group of elementary words. They are put right before the word they modify. They can't be modified. They can't be used alone. They are ignored when analyzing the grammatical structure of a compound. For example, if we apply a prefix to a noun, grammatically the whole word is still a noun.

Other rules

1. To build an antonym of an elementary word (i.e. change its meaning to the opposite), alternate the last vowels u/i. This can only be done where only one good objective solution exists, so this trick does not work for all elementary words.
